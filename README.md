# Stock Price Prediction System

## Overview
This project uses Machine Learning techniques to predict stock market price trends based on historical stock data. The goal is to analyze past market behavior and forecast future trends using data-driven insights.

## Features
- Historical stock data analysis
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning model training
- Trend prediction
- Data visualization using Matplotlib
- Model performance evaluation

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Dataset
The project uses approximately 3 years of historical stock market data containing:
- Open Price
- Close Price
- High Price
- Low Price
- Volume

## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Feature Engineering
4. Exploratory Data Analysis
5. Model Training
6. Prediction
7. Performance Evaluation

## Results
- Achieved approximately **82% directional prediction accuracy** on test data.
- Generated visual insights to identify market trends and patterns.
- Improved understanding of stock market behavior through predictive analytics.
